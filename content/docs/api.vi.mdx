# Truy cập API

ALumiEye cung cấp API cho gói Doanh nghiệp để tích hợp tín hiệu và dữ liệu vào hệ thống của bạn.

## Tổng quan

API cho phép:

- **Lấy tín hiệu** — Tín hiệu AI qua API
- **Dữ liệu thị trường** — Dữ liệu thời gian thực và lịch sử
- **Dòng tiền thông minh** — Dữ liệu dòng vốn tổ chức
- **Webhooks** — Cảnh báo khi có tín hiệu mới

## Xác thực

API dùng khóa API. Khóa được cấp sau khi kích hoạt gói Doanh nghiệp.

Liên hệ bộ phận kinh doanh để biết thêm chi tiết và tài liệu đầy đủ.
